7.C, 1.1.1- decorative-background-image

Test Name Test ID Test Condition
1.1.1- decorative-background-image 7.C The background image is not the only means used to convey important information.

Evaluate Results:

If any of the following is TRUE, the content PASSES:

  1. The background image is decorative,
    OR
  2. The meaning of the background image is also available without the background image.

Interactive Samples

  • Images Examples

Tips:

  • Because background images are often used in interactive elements (e.g., buttons and links), it can be helpful to use other ANDI modules (e.g., focusable elements and/or links/buttons), to identify whether information is conveyed with more than just the image itself.
  • As described in the "How to Test" portion of the test process for Test ID 7.C, use both the "find background" and "hide background" features of the ANDI: graphics/images module to identify background images and evaluate whether their equivalent meaning is available without the background images.

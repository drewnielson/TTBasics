5.B 2.4.6-label-descriptive

Test Name Test ID Test Condition
2.4.6-label-descriptive 5.B Each form label is sufficiently descriptive.

Evaluate Results:

If the following is TRUE, the content PASSES:

  1. Each form label is sufficiently clear and descriptive so users know what input data is expected.

Tips:

  • Do not include items that failed 5.A.
  • Enter data in each form element and try to submit to verify if there are data input requirements. (Important tip for training!)
  • If specific formats are required for data input, these must be conveyed in the form label (e.g., Date (MM/DD/YYYY)). Reliance on an error message is not sufficient.
  • This test may be somewhat subjective but, at minimum, the label should include enough information so a user understands the form's purpose and how to enter the requested data.

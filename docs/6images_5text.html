7.E, 1.4.5-image-of-text

Test Name Test ID Test Condition
1.4.5-image-of-text 7.E The image of text cannot be replaced by text or is customizable.

Evaluate Results:

If any of the following is TRUE, the content PASSES:

  1. The image of text cannot be replaced with text, OR
  2. The image of text can be visually customized.

Interactive Samples

  • Images Examples

Tips:

  • Images that contain text, but are excluded from this test, must still be evaluated against the other tests in section 7 Images.
  • EXCLUDE text that is part of a picture that contains significant other visual content.
  • Text in a logo image = PASS (not DNA), because Evaluate Results condition 1 is TRUE.
